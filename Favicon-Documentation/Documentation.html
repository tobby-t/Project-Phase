Introduction

FaviconGen is a webapp used to generate and download favicons for individuals personal or public use, which can be in the form of an icon or an embedded code.

What is a favicon?

Favicon or Favorite icon is a small pixel icon or a tiny image, located at the tab of a website or in a bookmark bar, that serves as a brand for a website, and allows users to identify websites when many tabs are open.

Format for Download

In FavIconGen a user has the option to download their Favicon in various formats. To satisfy all of the browser requirements, we have three sizes (16x16, 32x32, and 48x48) plus a .ico or .png. file for each of them. These are:

Embedded Link

Embed codes are a bunch of URLs (HTML codes) that render a file on the internet. So, an embed code for a Facebook favicon could be:

https://www.facebook.com/favicon/library/abcdef54256454

JPEG

JPEGs are simple image formats. They contain compressed image data

PNG

PNGs are also simple image formats. It is a raster image type which is very popular.

SVG/ICO

SVGs are Scalable Vector Graphics. This format is usually very easy to sca;e without compromising the pixel perfect state.In case something goes wrong, a “.ico” file is required to allow the browser to locate the backup icon.

Zipped File

A user can also download a zipped file containing all the formats for download.

How to use FaviconGen

FaviconGen gives you an easy way to download Favicons for future use either as an Icon,image or an embedded code or as a zip file that contains every format needed. There are three ways to download a favicon on FaviconGen:

  1. Generate Icon From Text

    FaviconGen gives you the option to generate your own favicon just by inputting a text.

    Steps to Generate Icons From Text

    • Go on the Website
    • Click on the Generate Icons Button on the Hero Section or on the footer
    • Choose Generate Icon from text
    • Input a Text that represents your brand
    • Choose a color
    • Choose a download format ( PNG, SVG, JPeg, Embedded Code or A Zip file containing all the formats)
    • Click on the Download Button
    • You are now ready to use your favicon


  2. Generate Icon from Image

    FaviconGen gives you the option to generate your own favicon just by uploading an image.

    Steps to Generate Icons From Image

    • Go on the Website
    • Click on the Generate Icons Button on the Hero Section or on the footer
    • Choose Generate Icon from Image
    • Click on the Upload Image
    • Choose an image that best suits your brand
    • Click on the Generate button
    • Choose a download format ( PNG, SVG, JPeg, Embedded Code or A Zip file containing all the formats)
    • Click on the Download Button
    • You are now ready to use your favicon

Features

There are various features in the WebApp for both Unauthenticated Users as well as Authenticated Users

Users who have an account and are logged in have complete access to the all features available, which are:

Unauthenticated Users

  • Visit the platform to view basic information about it
  • View and interact with the documentation

Generator

Users can create favicons from a text or image which can be converted into a zip file of various file types or an embedded HTML code.

Image-Based Generator:

For Developers

  • Basic way to set favicon.ico in HTML

    <link rel="shortcut icon" href="pictures/favicon2.ico" type="image/x-icon">
    This example would cause the browser to look for the favicon2.ico file and use it instead of the favicon.ico. If an older browser or a browser that does not support this feature is used, the tag is ignored.
    The favicon can also be pulled from another website. For example, to use the Computer Hope favicon on your site, you could add the tag below in your <head></head> section.

    <link rel="shortcut icon" href="https://www.computerhope.com/favicon.ico">

  • In JavaScript

    var favicon = require('serve-favicon');
    app.use(favicon(path.join(__dirname, 'public/images', 'favicon.ico')));

  • Basic way to set favicon.png in HTML

    <link rel="shortcut icon" href="../images/favicon.png" />
    This plain HTML code will runs with JavaScript, inside <head> tag.

  • In JavaScript

    function setFavicons(favImg){

    let headTitle = document.querySelector('head');
    let setFavicon = document.createElement('link');
    setFavicon.setAttribute('rel','shortcut icon');
    setFavicon.setAttribute('href',favImg);
    headTitle.appendChild(setFavicon);

    }

    setFavicons('https://spemer.com/img/favicon/favicon.png');
